This is a project worked on by 5 people including myself. 

*Overview:

TimeLink is a web application designed to help users visually compare their schedules with friends. The tool highlights overlapping events and conflicts, making it easier to coordinate meetups and manage time efficiently.

*Features:

User Registration & Login: Secure authentication to manage individual schedules.

Schedule Input & Management: Users can input and edit their schedules.

Friend Schedule Comparison: Compare schedules with added friends.

Visual Conflict Display: Highlights conflicting time slots for better planning.
